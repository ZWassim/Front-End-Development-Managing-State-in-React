# TaskMaster Todo Application

A modern, feature-rich Todo List application built with React that demonstrates advanced state management, form validation, and local storage persistence.

## Features

✨ **Complete CRUD Operations**
- Add new tasks with name and description
- Edit existing tasks
- Delete tasks with confirmation
- Mark tasks as completed

🎯 **Form Validation**
- Real-time validation for task name (3-100 characters)
- Real-time validation for description (5-500 characters)
- Clear error messages
- Prevents submission of invalid data

💾 **Data Persistence**
- Automatic save to localStorage
- Tasks persist between browser sessions
- Error handling for storage operations

🎨 **User Interface**
- Clean, modern design
- Visual distinction between active and completed tasks
- Responsive layout for all devices
- Smooth animations and transitions
- Hover effects and interactive elements

🛠 **Additional Features**
- Task counter (total and completed)
- Sorting (active tasks first)
- Creation timestamps
- Edit mode with cancel option
- Disabled edit for completed tasks
- Confirmation dialogs for deletions

## Technology Stack

- **React** (Hooks: useState, useEffect)
- **CSS3** with modern features (Flexbox, Grid, Animations)
- **LocalStorage API** for data persistence
- **Date API** for task timestamps
